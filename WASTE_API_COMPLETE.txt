## 📡 API Endpoints Explained

All endpoints require **Admin role** + **Authentication** + **CSRF protection** (for mutations)

### 1. **GET /api/waste-categories**
List all waste categories with pagination support
- **Returns:** Array of category objects
- **Status:** 200 OK
- **No body required**

### 2. **POST /api/waste-categories**
Create a new waste category
- **Required fields:** name, description, basePrice
- **Optional:** category_icon, hazardous
- **Status:** 201 Created
- **CSRF Token:** ✅ Required

### 3. **GET /api/waste-categories/{id}**
Get detailed information about a specific category
- **URL param:** id (integer)
- **Status:** 200 OK or 404 Not Found
- **No body required**

### 4. **PUT /api/waste-categories/{id}**
Update a waste category (partial update allowed)
- **URL param:** id (integer)
- **Body:** Any fields to update
- **Status:** 200 OK
- **CSRF Token:** ✅ Required

### 5. **DELETE /api/waste-categories/{id}**
Delete a waste category
- **URL param:** id (integer)
- **Status:** 200 OK or 409 Conflict (if in use)
- **CSRF Token:** ✅ Required

### 6. **GET /api/waste-categories/pricing**
Get all categories with dynamic pricing tiers
- **Returns:** Categories with pricing brackets
- **Status:** 200 OK
- **Query param:** include_stats (optional boolean)

## 📊 Data Model

```json
{
  "id": 1,
  "name": "Plastic",
  "description": "All types of plastic waste...",
  "basePrice": 50.00,
  "category_icon": "♻️",
  "hazardous": false,
  "created_at": "2025-11-29 08:30:00",
  "updated_at": "2025-11-29 08:30:00"
}
```

**Constraints:**
- `name` - Required, 1-100 chars, unique
- `description` - Required, 10-500 chars
- `basePrice` - Required, > 0, max 2 decimals
- `category_icon` - Optional emoji/icon
- `hazardous` - Optional boolean flag
